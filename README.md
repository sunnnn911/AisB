# 폐열 발전 실시간 모니터링 대시보드

Arduino UNO를 USB로 연결한 뒤, GitHub Pages 홈페이지에서 실시간으로 온도·전압·전류·전력을 확인하는 웹 대시보드입니다.

## 기능

- 아두이노 USB 실시간 연결
- 현재 뜨거운 면 온도, 차가운 면 온도, 온도차 표시
- 현재 전압, 전류, 전력 표시
- 실시간 그래프 표시
- 전압이 3.00V 이상이 되는 순간 화면 중앙에 ⚡ 효과 표시
- 온도차가 20℃ 이상이 되는 순간 화면 중앙에 🔥 효과 표시
- 팬이 ON으로 바뀌는 순간 🌀 효과 표시
- 데이터 저장 시작/중지
- Excel에서 열 수 있는 CSV 저장
- 새 실험 시작 버튼으로 표와 그래프 초기화

## 사용 방법

1. Arduino에 `arduino/TEG_Rotor_Experiment.ino` 코드를 업로드합니다.
2. 노트북/PC에서 Chrome 또는 Edge를 엽니다.
3. GitHub Pages 주소로 들어갑니다.
4. `[아두이노 연결]` 버튼을 누릅니다.
5. 포트 선택 창에서 Arduino UNO 포트를 선택합니다.
6. `[데이터 저장 시작]` 버튼을 누르면 표에 데이터가 쌓입니다.
7. `[Excel용 CSV 저장]` 버튼을 누르면 엑셀에서 열 수 있는 CSV 파일이 저장됩니다.

휴대폰이나 태블릿에서는 Web Serial 연결이 제한될 수 있습니다.
